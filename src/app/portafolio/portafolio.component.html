Aplicación MVC

Dos Proyectos Demo de una página web. uno es una pagina web para crear blogs y gestionarlos (blogsweb) y otro es una pagina web para gestionar un sistema de facturas y clientes (facturasweb)

Backend (Spring):

  • Desarrollo Backend con Spring Boot para gestionar blogs y usuarios en el proyecto (blogsweb), facturas y clientes en el proyecto (facturasweb).
  • Implementación de autenticación y autorización segura con Spring Security para proteger las sesiones de usuario.
  • Administración efectiva de datos mediante Spring Data (JPA) y una base de datos MySQL.

Frontend (Thymeleaf):

  • Creación de una interfaz de usuario dinámica y atractiva utilizando Thymeleaf.
  • Integración con el backend para mostrar y crear blogs, gestionar usuarios en el proyecto (blogsweb) y crear facturas, gestionar usuarios y clientes en el proyecto (facturasweb).
